# Esercizio Angular

Sviluppare una web-app per la gestione delle ricette, in stile "ricettario di nonna papera" o "giallo zafferano".

Mi aspetto che esistano almeno 2 modelli: Ricetta e Categoria.
Sviluppare tutte le viste necessarie per la visualizzazione delle ricette, navigazione per categoria, ricerca di ricette per titolo.

- Funzionalità Admin: può creare, modificare, eliminare ricette e categorie
- Funzionalità utente anonimo: può navigare il sito, leggere le ricette e fare ricerche
- Funzionalità utente registrato: può mettere una ricetta tra i preferiti
